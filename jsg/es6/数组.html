    # 数组

    # 一、Array方法：

    1.Array.isArray(): 判断是否是数组类型
2.Array.from(伪数组对象): 把一个伪数组转成数组
3.every() 方法测试数组的所有元素是否都通过了指定函数的测试
4.every里的函数中还有三个参数: 元素值,元素索引,原数组
5.some():有一个满足条件的就返回true
6.filter():返回一个新的数组,数组中包含所有通过测试函数的元素
7.map(): 创建一个新数组，结果是该数组中的每个元素都调用一个提供的函数后返回的结果
8.join("|"):指定一个字符串来分隔数组的每个元素
9.push(要插入的元素): 往数组的最末尾插入一个元素,返回更改后的数组的长度
10.pop():删除数组中最末尾的元素,并返回被删除的元素
11.shift():删除了数组中的第一个元素并且返回了被删除的元素
12.unshift():往数组的第一个位置插入元素并返回更改后的数组的长度
13.resever(): 把数组元素进行反转,该方法会改变原数组
14.sort():对数组的元素进行排序，不稳定性  封装方法
15.splice( 指定修改的开始位置,要被删除的元素个数,要插入的元素):
16.concat() 连接两个或更多的数组，并返回结果。
17.slice() 从某个已有的数组返回选定的元素
18.toSource() 返回该对象的源代码。
19.toString() 把数组转换为字符串，并返回结果
20.toLocaleString() 把数组转换为本地数组，并返回结果。
21.valueOf() 返回数组对象的原始值

    # 二、Array属性：

    1.constructor 返回对创建此对象的数组函数的引用。
2.length 设置或返回数组中元素的数目。
3.prototype 使您有能力向对象添加属性和方法。

    # 三、Array实例：

    # 定义：用一个变量来储存一系列值

    # 作用：存储多个数据

    # 创建方式：

    1.通过构造函数的方式创建：只要使用了new，就会在堆内存中申请空间

    1.定义一个变量
var a = 10;
2.创建一个空数组
var arr = new Array();
console.log(arr)//[]
3.取名要做到见名知意，如果Array括号里给一个值，该数值代表数组的长度
var arr = new Array(2);
console.log(arr);//2
   数组名.length：数组的长度
4.Array给了多个值，这些值会作为数组的元素存放于数组
var arr = new Array(1,2,3,4,5,6);
console.log(arr);//[1,2,3,4,5,6]

    2.通过字面量的方式创建

    1.数组是如何设置值和取值的：通过下标(索引)来设置值和取值的
元素：数组中的每一个元素
索引：从0开始，依次递增
最大索引和数组的长度的关系：最大索引= 数组的长度-1
var arr = [22,33,44,55];
// 获取
console.log(arr[2]);//44
// 设置
arr[2] = 66;
console.log(arr[2]);//66
2.数组可以存储不同的类型吗？可以
var arr = ['study',12,undefined,null];
console.log(arr);

    # 四、Array举例：

    1.concat：用于合并两个或多个数组，此方法不会改变现有数组，而是返回一个新数组

    var arr1 = [1,2,3,4,5];
var arr2 = [6,7,8,9,10];
var newArr = arr1.concat(arr2);
console.log(newArr);

    2.push：将一个或多个元素添加到数组的末尾，并返回该数组的长度

    var arr = [123,456,789];
var len = arr.push(88);
console.log(len);
console.log(arr);

    3.unshift();将一个或多个元素添加到数组的开头，并返回该数组的长度

    var arr = [123,456,789];
var len = arr.unshift(88);
console.log(len);
console.log(arr);

    4.pop：从数组中删除最后一个元素，并返回该元素的值

    var arr = [123,456,789];
var len = arr.pop();
console.log(len);
console.log(arr);

    5.shift：从数组中删除第一个元素，并返回该元素的值

    var arr = [123,456,789];
var len = arr.shift();
console.log(len);
console.log(arr);

    6.sort：数组排序 默认通过ASCII排序， 对数字排序不好使，所以sort提供一个可编程接口

    arr.sort(function(a,b){return a - b;});
arr.sort(function(a,b){return b - a;});
a - b正序/升序
b - a倒序/降序

    7.join:数组转成字符串

    var arr = [123,223,456,556];
console.log(arr.join('-'))
