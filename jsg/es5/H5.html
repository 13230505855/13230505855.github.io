# H5

# 一、H5标签：

<header></header>    头部信息 唯一   移动端最小字号 font-size：13px；
<section></section>  区域段节  范围比div大，语义比div更强，可以包含header、h1-h6……凸显语义的标签
<article></article>  文章标签  可以是文章、blog、帖子、短文或者回复、评论
<aside></aside>      侧边栏
<footer></footer>    底部信息 唯一

# 二、音频

必须加controls="controls" 属性才能显示播放插件
<source src="音频路径"></source>用source链接音频路径
   1.<audio  controls="controls" >
	<source src="">
	</source>
      </audio>		

   2.<audio src="音频地址"></audio>
元素支持的3种文件格式：MP3、Wav、Ogg

# 三、音频

目前必须加controls="controls" 属性才能显示播放按钮
<source src="视频路径"></source>用source链接音频路径
   1.<audio  controls="controls"  loop="loop">
	<source src=""></source>
      </audio>		
   2.<audio src="音频地址"></audio>

# 四、属性

controls 控制播放和暂停 
loop 循环播放
autoplay 自动播放
preload 页面加载时是否加载音频/视频 auto默认/metadata加载/none不加载
muted 静音播放
poster 封面
autoplay与preload任选其一
